=== Deploying Models ===

Option 1: Declarative (recommended for production)
  Bake model code into a custom image, then set in values:
    image:
      repository: your-registry/your-ray-image
      tag: latest
    serveApplications:
      - name: my-model
        route_prefix: /predict
        import_path: myapp.model:app

Option 2: From a Jupyter notebook
  import ray
  from ray import serve
  import requests

  ray.init("ray://{{ include "nebari-rayserve.head-service-name" . }}.{{ .Release.Namespace }}.svc.cluster.local:10001")

  @serve.deployment
  class Hello:
      async def __call__(self, request):
          return "Hello from Ray Serve!"

  serve.run(Hello.bind(), name="hello", route_prefix="/hello")

  resp = requests.get("http://{{ include "nebari-rayserve.serve-service-name" . }}.{{ .Release.Namespace }}.svc.cluster.local:8000/hello")
  print(resp.text)

NOTE: Python and Ray versions in your notebook must match the cluster.
This chart uses Ray {{ .Chart.AppVersion }} with Python 3.9.
